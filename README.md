A secure Password Manager application built to store, manage, and retrieve user credentials safely. The application provides authentication, encrypted password storage, and CRUD operations for managing saved accounts.

Features
User Registration and Login
JWT Authentication
Two-factor Authentication
Secure Password Hashing using bcrypt
Store Website Credentials
Add New Passwords
View Saved Passwords
Update Existing Passwords
Delete Passwords
Search Credentials
RESTful APIs
Environment Variable Configuration
Secure Backend Architecture
